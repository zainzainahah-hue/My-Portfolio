# 🚀 Personal Portfolio & Project Showcase

Welcome to my personal profile website. This project is a clean, responsive, and animated portfolio designed to showcase my skills and academic projects.

## 🛠 Built With
This website is built using pure frontend technologies without any JavaScript, focusing on high-quality CSS animations and clean HTML structure:
* HTML5: Semantic structure for better accessibility.
* CSS3: Advanced styling, Flexbox layout, and custom Keyframe animations.
* No JavaScript: Optimized for speed and simplicity.

## 🌟 Key Features
* Custom Animations: Smooth hover effects and continuous rotation for icons.
* Responsive Design: Optimized for both desktop and mobile viewing.
* Project Integration: Direct links to my C++ Pharmacy Management System.
* Professional Layout: Clear sections for personal info, skills, and contact.
